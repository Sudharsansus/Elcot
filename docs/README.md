# AVGC-XR Portal - Documentation

## Overview

This directory contains all project documentation for the Tamil Nadu AVGC-XR Portal (ELCOT Tender ELCOT-ST-FPP-2026-0026).

## Directory Structure

| Directory | Description |
|-----------|-------------|
| `architecture/` | System architecture, ADRs, C4 diagrams, sequence diagrams |
| `api/` | OpenAPI specifications and API documentation |
| `compliance/` | GIGW, WCAG 2.1 AA, DPDP Act 2023, ISO 27001, tender eligibility |
| `onboarding/` | Developer onboarding guides |
| `operations/` | Deployment, scaling, disaster recovery, rollback, cost optimisation |
| `runbooks/` | Incident response runbooks for each infrastructure component |
| `security/` | Threat model, secure coding, secret management, incident response |

## Documentation Standards

- All documentation in English (primary) with Tamil translations where applicable
- Markdown format for all prose documents
- ADRs follow the Michael Nygard format
- Runbooks follow the Google SRE template
- Diagrams use Mermaid or PlantUML syntax

# GIGW Compliance Checklist

## Guidelines for Indian Government Websites

The AVGC-XR Portal complies with the **Guidelines for Indian Government Websites (GIGW) 3.0** as mandated by the Ministry of Electronics and Information Technology (MeitY).

## Compliance Matrix

| Requirement | Status | Evidence |
|-------------|--------|----------|
| Bilingual content (Hindi/regional + English) | Compliant | Tamil + English throughout all portals |
| Unified header with government emblem | Compliant | Tamil Nadu government header component |
| Feedback mechanism | Compliant | Feedback form in footer |
| Accessibility (WCAG 2.0 Level A minimum) | Compliant | WCAG 2.1 AA achieved (exceeds GIGW minimum) |
| Copyright policy | Compliant | Footer copyright notice |
| Privacy policy link | Compliant | Footer privacy policy page |
| Terms of use | Compliant | Footer terms of use page |
| Sitemap | Compliant | Auto-generated sitemap.xml |
| Search functionality | Compliant | Elasticsearch-powered search with Tamil support |
| Mobile responsive | Compliant | Responsive design with design system breakpoints |
| SSL/TLS encryption | Compliant | TLS 1.2+ enforced on all endpoints |
| Page load time < 5 seconds | Compliant | Angular lazy loading, CDN assets, gzip compression |
| Contact information | Compliant | Contact page with ELCOT office details |
| Content freshness | Compliant | Last updated dates on all CMS-managed pages |
| Navigation consistency | Compliant | Shared navigation components across portals |
| Standard fonts | Compliant | Noto Sans SC (Tamil), Inter (English) |
| Meta tags and descriptions | Compliant | SEO meta tags on all pages |

## GIGW Score Target

Minimum score: 85% (GIGW compliance)
Target score: 95%+ (best practice)

## Testing

GIGW compliance is verified using the online GIGW self-assessment tool before each release.

# AVGC-XR CMS (Strapi)

Headless CMS for managing bilingual content (English/Tamil).

## Content Types
- **Scheme** — Government scheme details, guidelines, FAQs
- **Content Page** — Static pages (About, Contact, Help)
- **Event** — AVGC industry events and announcements
- **FAQ** — Frequently asked questions per scheme

## Development
```bash
cd apps/cms
npm install
npm run develop
```
## Admin Panel
http://localhost:1337/admin
